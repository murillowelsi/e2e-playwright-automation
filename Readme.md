# Basic Playwright E2E Test With Typescript

## 🚀 Tech

- [Node.js]
- [TypeScript]
- [Playwright]

## 👨🏻‍💻 How to execute

[Node.js](https://nodejs.org/) v16 or higher

Install Playwright

```sh
npx playwright install
```

Install dependencies

```sh
yarn install
```

Run tests

```sh
yarn test
```

Generate reports

```sh
yarn test:report
```
